# Car Sales Analytics Dashboard (Power BI)Car Sales Analytics Dashboard (Power BI)

A Power BI dashboard built using a 25,000-row car sales dataset to analyze sales performance, pricing trends, and year-over-year growth.

![Dashboard Preview](https://raw.githubusercontent.com/yourusername/yourrepo/main/your-image.png)

### **What This Project Covers**
- Data loading from Excel & basic cleaning using Power Query
- Calendar table creation & proper data modeling (star schema)
- Core KPIs: Total Sales, Avg Selling Price, Units Sold
- Time intelligence KPIs: YTD, MTD, YoY, Difference vs Previous Period

### **Key Visuals** 
- Weekly Sales Trend with DAX-based peak highlight
- Sales by Body Style & Color (Donut Charts)
- Cars Sold by Region (Map)
- Company-wise performance table with shares & YTD metrics

### **DAX Skills Used**
- TOTALYTD, TOTALMTD, SAMEPERIODLASTYEAR
- Conditional formatting (green/red)
- MAXX + IF logic for dynamic highlights

### **Business Insights**
- Top brands, regions, and dealers
- Weekly & yearly sales trends
- Pricing and volume performance
- Color, body style, and company contribution analysis
